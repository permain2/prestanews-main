Understanding Cash Back Rates

Types of Cash Back Structures

Flat-Rate Cards Earn the same percentage on everything:

  • 1.5% to 2% on all purchases
  • Simple to use‚Äîno category tracking
  • Best for people who want simplicity

Category Bonus Cards Earn more in specific spending categories:

  • 3-6% on groceries, gas, dining, etc.
  • Requires using right card for each purchase
  • Higher earning potential with effort

Rotating Category Cards Bonus categories change quarterly:

  • 5% on activated categories
  • Requires quarterly activation
  • Categories like Amazon, gas, restaurants rotate

Common Mistakes to Avoid

1. Carrying a Balance

Cash back is worthless if you pay interest:

  • 20% APR destroys any 2-6% cash back
  • Always pay in full
  • Set up autopay

2. Chasing Bonuses Over Value

Don‚Äôt overspend for rewards:

  • Only buy what you‚Äôd buy anyway
  • Sign-up bonuses shouldn‚Äôt drive unnecessary spending

3. Using Wrong Card

Missing out on bonus categories:

  • Know which card for each category
  • Label cards or use wallet organization
  • Set default cards by merchant on phone

4. Ignoring Shopping Portals

Leaving money on the table:

  • Check portals before every online purchase
  • Browser extensions automate this
  • Extra 1-15% adds up

5. Forgetting to Activate

Quarterly category activation:

  • Set calendar reminders
  • Activate day categories go live
  • Check all rotating category cards
